                DADU - Little utility 10  (LU10)
                        v1.0  19/04/92


This utility count the number of resets you do,
and write the  day and number of resets done  in a file
named: 'LU10.DAT'

LU10 should be used with a clock.
Put LU10_1.PRG in the auto folder of your disk.
